# TagNumeral.html
El programa para asignarle a los numerales arábigos etiquetas síntacticas y léxicas basadas en la gramática de los numerales en castellano


<b>Etiquetas (haz clic para expandir y ver todas las etiquetas)</b>
<i><u>uno; dos; tres</u></i> - número de digitos que contiene un numeral<br/>
<i><u>regular; irregular</u></i> - si el numeral contiene un cambio fonológico en comparación con el dígito de base (i.e. nueve > novecientos) - la irregularidad puede ser un cambio de consonante (cin- > quin- en quinientos), una elisión de consonante (-cientos > -ientos en quinientos) o un cambio vocálico (diptongo > vocal como en nueve > novecientos<br/>
<i><u>cero; no-cero</u></i> - Si el numeral contiene un <i>[0:tens]</i> en su number word frame <br/>
<i><u>teens; no-teens</u></i> - Si el numeral contiene un <i>[_:teens]</i> en su number word frame <br/>
